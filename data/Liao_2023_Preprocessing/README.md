# Liao 2023 Preprocessing

This is a partial copy of the [interpretable-splicing-model](https://github.com/liesese/interpretable-splicing-model) repository, retaining only the preprocessing scripts needed to reproduce the sequences and measurements from raw data for the Liao et al. 2023 paper.

# Usage

`preprocess.sh` takes the raw splicing analysis files from [GEO](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE200096) and converts them into a training and testing dataset. Raw and pre-processed datasets are stored under the `fasta_files` folder.
